Objectives
This project consists in creating a web forum that allows :

- communication between users.
- associating categories to posts.
- liking and disliking posts and comments.
- filtering posts.

SQLite
In order to store the data in your forum (like users, posts, comments, etc.) 
you will use the database library SQLite.

To structure your database and to achieve better performance
https://www.smartdraw.com/entity-relationship-diagram/

- You must use at least one SELECT, one CREATE and one INSERT queries.

Authentication

- client must be able to register as a new user on the forum
- create a login session to access the forum and be able to add posts and comments.
